Detailed Description

This class is an image viewer class that allows the user to sequence through a series of images. Using the VCR-like controls, the user can mark a start image and an end image and cycle through the images in between, automatically, in an animated fashion.

To use this class, set a child widget to be the viewer's main widget.

For example, if the parent widget is:

DockWidgetParent, call:

viewerWidget = nmfViewerWidget(...)
DockWidgetParent->setWidget(viewerWidget->getMainWidget());.
