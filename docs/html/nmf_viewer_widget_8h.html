class  nmfViewerWidget
 This class is an image viewer class that allows the user to sequence through a series of images. Using the VCR-like controls, the user can mark a start image and an end image and cycle through the images in between, automatically, in an animated fashion.

To use this class, set a child widget to be the viewer's main widget.

For example, if the parent widget is:

DockWidgetParent, call:

viewerWidget = nmfViewerWidget(...)
DockWidgetParent->setWidget(viewerWidget->getMainWidget());. More...
 

Detailed Description

GUI definition for the Image Viewer set of widgets.

This file contains the definition for the screenshot image viewer widget.

This file contains the GUI definitions for the controls and widgets use in the Image Viewer class. This class allows the user to view images one by one or set a begin and end image and flip through them in an animated fashion.
